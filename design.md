# 競合分析アプリ - デザインドキュメント

## アプリ概要
URLを入力して競合サイトのトラフィック、エンゲージメント、チャネル分析、キーワードデータを取得・比較できるモバイルアプリ。DataForSEO API、Google PageSpeed Insights API、SimilarWeb Data APIを統合。

## スクリーン一覧

### 1. ダッシュボード（ホーム）
自社サイトと登録済み競合サイトのサマリーを一覧表示。主要KPI（月間訪問数、直帰率、滞在時間、ページビュー）をカード形式で表示。

### 2. 競合サイト追加画面
URLを入力して新しい競合サイトを登録。自社サイトの設定も可能。

### 3. サイト詳細分析画面
選択したサイトの詳細データ（トラフィック推移、チャネル別内訳、エンゲージメント指標）を表示。

### 4. 比較画面
複数サイトを並べて比較。エンゲージメント指標の比較テーブル、アクセスシェア率のチャート。

### 5. 設定画面
APIキー管理、データ更新頻度設定。

## 主要コンテンツと機能

### ダッシュボード
- 登録サイト一覧（カード形式）
- 各サイトの月間訪問数・直帰率のスナップショット
- 「分析を更新」ボタン
- 「サイト追加」FABボタン

### 競合サイト追加
- URL入力フィールド
- サイト名入力
- 自社/競合の区分選択
- 追加ボタン

### サイト詳細分析
- トラフィック概要カード（月間訪問数、ユニーク訪問者、ページビュー）
- エンゲージメント指標（滞在時間、直帰率、ページ/訪問）
- チャネル別トラフィック内訳（棒グラフ）
- 月別トラフィック推移（折れ線グラフ）
- PageSpeed Insightsスコア

### 比較画面
- サイト選択チップ（複数選択可）
- 比較テーブル（エンゲージメント指標）
- アクセスシェア率（円グラフ）
- チャネル別比較（グループ棒グラフ）

## キーユーザーフロー

### フロー1: サイト追加→分析
1. ダッシュボードの「+」ボタンをタップ
2. URLとサイト名を入力
3. 「追加」をタップ
4. API経由でデータ取得（ローディング表示）
5. ダッシュボードに新サイトが追加される

### フロー2: 詳細分析の閲覧
1. ダッシュボードのサイトカードをタップ
2. サイト詳細分析画面に遷移
3. 各種グラフ・指標を閲覧
4. スクロールで全データを確認

### フロー3: 競合比較
1. 比較タブをタップ
2. 比較したいサイトを選択（チップ）
3. 比較テーブルとチャートが表示
4. 指標ごとにソート可能
